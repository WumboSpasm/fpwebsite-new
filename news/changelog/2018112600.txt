Flashpoint 5.2 - 'Type III'

The launcher has been given a makeover.
 * The new Home tab is now the start point, and contains several useful links and multiple randomly picked games.
 * The Browse tab now contains the playlists and game browse window.
 * Games can be added to playlists by dragging and dropping them onto the playlist you want to add them to.
 * Games can be removed from playlists with the new stop sign button when a game is selected in a playlist.
 * You can add icons to playlists by clicking on the question mark.
 * Playlist notes are now shown in the sidebar.
 * You can click on a screenshot to see it in a bigger size, and then click the bigger size to see it almost fullscreen.

(Note; some of these changes has required Theatre to stay on an old launcher build for the time being.)

Added playlists:
 * Favorites. An empty by default playlist that you can add games to.

 * Best of Nitrome by DarkMoe
