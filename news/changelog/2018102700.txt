Flashpoint 5.1 - 'The Great Filter'

The main new feature for Flashpoint Launcher is playlists; collections of games curated from the massive library in Flashpoint. Multiple playlists have been included with Arcade, and you can make your own to submit for inclusion. See the readme for more information.

Flashpoint Infinity has been split into three 'stages' to help reduce its size. See the readme for more information.

The launcher has also received several tweaks, updates and bugfixes.

Added support for the not-oft-used 3DVia Player plugin.

We removed a bunch of .pdb files from the Unity Web Player. It should save some space.

Controller support has been removed, due to a general lack of flexibility.
