We now support Java games.

We now have a batch file set that can back up your Flash save games and restore them.

About 50 games have been fixed for Flashpoint Infinity due to miscapitalized launch commands.

To save as much space as possible (mainly for Infinity), some space saving measures have been taken:
* Logos and screenshots have been compressed using PNGQuant. (Ones that weren't PNG have been converted to PNG.) This might cause some issues with logos and screenshots.
* Old projectors that weren't actively being used have been removed.
* Pale Moon's temporary storage has been cleared.

* Added the ArmorGames version of Gemcraft Zero, which has special powerups locked to it.
* Added a bunch of new logos to older games.
* Fixed the VVVVVV Demo.
* Fixed a couple of swapped screenshots and logos.
* Fixed credits on some of the games Terry Cavanagh worked on, at his request.
* Fixed name of Speed Racer to add the subtitle and demo status.
* Fixed Bubble Tanks TD 1.5's name.
* Fixed MoFro's genre.
* Fixed Jinx 3's name.
* Fixed publisher for two Garfield games.
