Flashpoint 6.0 - 'Alignment'

Major notes:

* Rearranged Flashpoint's directory structure for cleanliness. This helps the Redirector not pick up irrelevant software, has a neater file structure, and has a few other benefits we won't go into here.
* Pale Moon has been removed from Flashpoint, as all four of its games were transferred to running via Basilisk.
* We've added support for ActiveX games via a little known browser called The World. Yes, the Internet Explorer ActiveX. There's over a dozen games to check out.
* Our server now supports AMF; the main improvement in 6.0 is ArmorGames API support, which leads to a lot of new custom level databases in both newly included and old games.
* A bunch of the software that powers Flashpoint has been updated; Flash Player Projector version 32 is now the default, PHP has been updated to 7.2.14, and Apache has been updated to 2.4.38.
* There's a lot new in this version. I've probably forgotten something.
